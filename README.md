# MyWikiHelper
It's a chrome extension.
Port wiki text to http://*.huiji.wiki

## Feature
目前仅支持搬运至http://lotr.huiji.wiki
前提：
* 你在http://lotr.huiji.wiki 登录，且有编辑权限
* 目标页面不存在
